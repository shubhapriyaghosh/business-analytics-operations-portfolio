# Analytics & Operations Project 3
## Demand Forecasting & Capacity Planning Model for Operations Teams

### Purpose
This project simulates an operations planning model that forecasts case volume and translates demand into required staffing capacity, utilization and SLA risk.

### What is included
- Synthetic historical weekly demand dataset.
- Formula-driven forecast model using moving-average and scenario multipliers.
- Capacity plan converting forecasted cases into required FTE.
- SLA risk dashboard.
- Staffing and automation action plan.
- Source sheet with plain-text URLs.

### How to use
Open the Excel workbook and review:
1. `Overview`
2. `Inputs`
3. `Historical_Demand`
4. `Forecast_Model`
5. `Capacity_Plan`
6. `SLA_Risk_Dashboard`
7. `Staffing_Action_Plan`
8. `Sources`

### Privacy
All data is synthetic. No real operational, employee or customer data is used.
