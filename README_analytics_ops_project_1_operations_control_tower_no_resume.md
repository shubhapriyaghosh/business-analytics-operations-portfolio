# Analytics & Operations Project 1
## Operations Control Tower Dashboard — SLA, TAT and Backlog Monitoring

### Purpose
This project simulates a business operations control tower for monitoring case volume, SLA adherence, turnaround time, backlog aging, rework, escalation and team productivity.

### What is included
- Synthetic weekly operations data across 6 teams.
- Formula-driven executive dashboard.
- Team-level performance summary.
- Root-cause analysis.
- 90-day improvement roadmap.
- Source sheet with plain-text URLs.

### How to use
Open the Excel workbook and review:
1. `Overview`
2. `Assumptions`
3. `Raw_Data`
4. `Dashboard`
5. `RCA`
6. `Improvement_Plan`
7. `Sources`

### Privacy
All data is synthetic. No real customer, employee or company data is used.
